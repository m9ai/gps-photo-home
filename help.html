Help & Support

Getting Started

GPS-Photo is a Chrome extension that lets you add GPS metadata to your JPEG photos directly in your browser. No upload, no server — everything happens locally.

How to Use

  1. Select a photo: Click the file input to choose a JPEG photo from your device.
  2. Set GPS coordinates: You can either click on the interactive map to pick a location, or manually enter latitude and longitude values.
  3. Add extra metadata (optional): Fill in altitude, direction, speed, and timestamp if needed.
  4. Write & Download: Click the "Write GPS & Download" button to save the updated photo.

Batch Processing (Pro)

With a Pro license, you can process up to 50 photos at once:

  1. Switch to Batch mode.
  2. Select multiple JPEG photos.
  3. Set the GPS coordinates (same location will be applied to all photos).
  4. Click Batch Write GPS & Download to get a ZIP file with all processed photos.

Location Templates (Pro)

Save frequently used coordinates as templates for quick access. Click Save next to a template name to store the current coordinates. Click any saved template chip to load its coordinates instantly.

Activating Pro

  1. Purchase a license on Gumroad (Monthly, Yearly, or Lifetime).
  2. Copy the license key from your Gumroad receipt.
  3. Open the GPS-Photo extension popup.
  4. Click Upgrade to Pro, select your license type, paste the key, and click Activate.

Troubleshooting

My photos don't show GPS data after writing

Make sure you are viewing the photo in an app that supports EXIF GPS metadata. Some apps may not display GPS data. Try opening the photo in Google Photos, Apple Photos, or an EXIF viewer.

The map is not loading

The map requires an internet connection to load tiles from CartoDB. Please check your network connection and try again.

License activation failed

Make sure you selected the correct license type (Monthly, Yearly, or Lifetime) that matches your purchase. If the issue persists, contact Gumroad support with your order details.
